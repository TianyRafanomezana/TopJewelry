# TopJewelry

Objets 3D et Interfaces Web
Projet MMI

## 1 Descriptif du projet
Dans ce projet, nous vous demandons de développer un site web de présentation d’un objet en trois dimensions.
Ce site aura un objectif d’apprentissage, ou de découverte de l’objet en question.
Il peut s’agir d’un objet 3D complexe (e.g. un assemblage de pièces mécaniques, ou une anatomie végétale ou
animale, ...), d’un environnement 3D avec différents éléments (e.g. un bâtiment, un musée, un parc d’attractions, ...), ou d’un concept abstrait ou une schématisation (e.g. structure hiérarchique, schéma d’assemblage, ...).
L’utilisateur devra pouvoir interagir avec l’objet (manipulations, navigations, animations, ...) et certaines
parties seront liées à des éléments informatifs de type texte, images, ou vidéos, intégrés dans la page. De
même, l’interaction avec certains éléments informatifs se répercutera sur l’objet 3D.

## 2 Objectif du projet
Ce projet a pour objectif de vous ouvrir à l’utilisation d’éléments 3D dans un environnement web. Vous exploiterez les possibilités offertes par ce mode de visualisation et d’interaction, ainsi que les liens possibles avec les
environnements classiques (pages web 2D scrollables)

## 3 Technologies et ressources
Le site web ne comportera qu’une seule page, dans laquelle vous utiliserez les technologies web classiques que
vous connaissez (html, javascript, css).
La 3D sera gérée avec le moteur BabylonJS : https ://www.babylonjs.com.
Pour vous aider, BabylonJS met à disposition un playground en ligne : https ://playground.babylonjs.com.
Dans un premier temps il vous permettra de récupérer une structure basique du moteur. Ensuite, il pourra vous
servir à expérimenter et/ou à générer automatiquement quelques lignes de code pour les objets et les actions les
plus simples.
Vous pourrez générer des objets 3D avec un logiciel de CAO si vous en connaissez, mais cela prend du
temps et il existe de nombreux sites web permettant d’en télécharger gratuitement, comme par exemple :
https ://sketchfab.com/3d-models (inscription gratuite).
Vous pouvez utiliser des librairies (javascript par exemple), mais nous devons pouvoir ouvrir le projet (la page
web) dans un navigateur sans installation préalable !

## 4 Déroulé du projet
La réalisation du projet se fera sur les séances de TP dédiées. Ces séances débuteront par une courte présentation
des éléments de base du domaine de l’ ‘Interaction 3D’ et du moteur 3D ‘BabylonJS’.
Il est possible de travailler et rendre le projet en monôme ou binôme uniquement. Dans le cas de binôme, il
devra être déclaré et stable dès le deuxième jour de TP.

## 5 Evaluation
Vous serez évalué(s) en particulier sur :
• les interactions mises en place pour accéder aux différentes parties de l’objet 3D
• l’efficacité et la qualité des liens entre la 3D (ex : l’objet) et la 2D (ex : les éléments informatifs)
• l’utilisabilité/UX générale du site
• l’effort d’originalité et de cohérence de l’ensemble (idée, présentation, code, ...)
Pour chaque item ci-dessus, la qualité et la quantité seront évaluées conjointement.
